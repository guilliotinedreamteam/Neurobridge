# Tech Stack: NeuroBridge

## 1. Backend Layer
- **Language:** Python 3.10+
- **Web Framework:** FastAPI (Asynchronous API)
- **Machine Learning:** TensorFlow (Neural Signal Decoding)
- **Data Science:** NumPy, SciPy (Signal Processing)
- **Logging:** Loguru

## 2. Frontend Layer
- **Language:** TypeScript
- **UI Framework:** React 18+
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Component Library:** Radix UI, Lucide React
- **Data Visualization:** Recharts (D3-based)

## 3. Communication & Data
- **Real-time:** WebSockets (FastAPI + React)
- **Protocols:** JSON (Typed), Asyncio-driven Producer-Consumer engine

## 4. Infrastructure & Quality
- **Version Control:** Git
- **Package Management:** npm (Frontend), pip (Backend)
- **Testing:** Pytest (Backend), Vitest (Frontend candidate)
